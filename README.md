  <ol style="margin:10px 0 12px 20px;color:#0f172a;">
    <li><strong>Best browser:</strong> Use <em>Google Chrome (desktop)</em> for the most reliable experience. Android Chrome works too. Safari on iOS has limited voice recognition.</li>
    <li><strong>How to use:</strong> Click <strong>Start Listening</strong>, ask a question out loud (see examples below), wait for the bot to speak, and use <strong>Replay Last Answer</strong> if needed.</li>

    <li><strong>Mic permission:</strong> When the browser asks for microphone permission, click <em>Allow</em>. If nothing happens, refresh the page and allow mic access again.</li>
  </ol>

  <p style="margin:0 0 8px;color:#334155;"><strong>Troubleshooting</strong></p>
  <ul style="margin:6px 0 12px 20px;color:#0f172a;">
    <li>If the bot does not hear you: check OS/browser mic settings and ensure the browser has permission to use the mic.</li>
    <li>If recognition fails repeatedly: try Chrome desktop, click Stop, refresh the page, then Start Listening again.</li>
    <li>If speech does not play: ensure your speakers/headphones volume is on and not muted. Some browsers may require a user gesture (click) before playing audio.</li>
  </ul>
